## DEVELOPMENT WORKFLOW

1. Clone/download project
2. Run: npm install (both root and server/)
3. Setup Supabase:
   - Create account at supabase.com
   - Create database
   - Run SQL schema
4. Create .env files with credentials
5. Start backend: cd server && npm run dev
6. Start frontend: npm run dev
7. Access at http://localhost:5173
